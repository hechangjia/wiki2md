# wiki2md v0.3 Evidence Layer Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add section-level evidence mapping that preserves explicit citation provenance in sidecars while keeping `article.md` clean.

**Architecture:** Extend normalization to capture citation ids per section before visible prose cleanup removes them, aggregate those ids against existing `ReferenceEntry` records, and emit two new bundle artifacts: `section_evidence.json` and `sources.md`. Keep rendering of `article.md` unchanged in style and treat evidence as a parallel layer rather than reinserting footnotes into prose.

**Tech Stack:** Python 3.12+, `pydantic`, `BeautifulSoup4`, `pytest`, `pathlib`, `json`, `ruff`, `uv`

---

## File Structure

- `src/wiki2md/document.py`: extend document models with section-aware evidence types
- `src/wiki2md/normalize.py`: capture citation ids from section content and aggregate section-level evidence
- `src/wiki2md/render_sources.py`: render human-readable `sources.md` from section evidence

```python
def _extract_node_reference_ids(node: Tag) -> list[str]:
    ids: list[str] = []
    seen: set[str] = set()
    for anchor in node.select('sup.reference a[href], sup.mw-ref a[href]'):
        href = anchor.get("href", "")
        if not href.startswith("#"):
            continue
        ref_id = href.removeprefix("#").strip()
        if not ref_id or ref_id in seen:
            continue
        seen.add(ref_id)
        ids.append(ref_id)
    return ids


def _slugify_section_id(text: str) -> str:
    return re.sub(r"[^a-z0-9]+", "-", text.casefold()).strip("-") or "section"
```

```python
def _finalize_section(section: SectionEvidence, references: list[ReferenceEntry]) -> SectionEvidence:
    by_id = {reference.id: reference for reference in references if reference.id}
    sources: list[SectionEvidenceSource] = []
    primary_urls: list[str] = []
    for ref_id in section.reference_ids:
        reference = by_id.get(ref_id)
        if reference is None:
            continue
        source = SectionEvidenceSource(
            id=reference.id,
            text=reference.text,
            primary_url=reference.primary_url,
            link_kinds=[link.kind for link in reference.links],
        )
        sources.append(source)
        if reference.primary_url and reference.primary_url not in primary_urls:
            primary_urls.append(reference.primary_url)
    section.sources = sources
    section.reference_count = len(section.reference_ids)
    section.primary_urls = primary_urls
    return section
```

After references are collected, finalize all sections and assign:

```python
    document.section_evidence = [
        _finalize_section(section, document.references) for section in section_evidence
    ]
```

```python
from wiki2md.document import SectionEvidence


def render_sources_markdown(title: str, sections: list[SectionEvidence]) -> str:
    lines = [f"# Sources for {title}", ""]
    for section in sections:
        lines.append(f"{'#' * max(section.level, 2)} {section.heading}")
        lines.append("")
        if not section.sources:
            lines.append("_No explicit sources mapped._")
            lines.append("")
            continue
        for source in section.sources:
            line = f"- {source.text}"
            if source.primary_url:
                line += f" ({source.primary_url})"
            if source.link_kinds:
                line += f" [{', '.join(source.link_kinds)}]"
            lines.append(line)
        lines.append("")
    return "\n".join(lines).rstrip() + "\n"
```
