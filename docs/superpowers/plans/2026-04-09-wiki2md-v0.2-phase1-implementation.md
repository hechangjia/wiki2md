# wiki2md v0.2 Phase 1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add infobox extraction, `infobox.json`, richer image metadata fallbacks, and a readable `## Profile` section so each saved person page becomes a more complete corpus unit.

**Architecture:** Extend the current `Document` model with an explicit infobox structure, keep body blocks focused on prose/images/sections, and thread the new infobox data through normalization, asset selection, Markdown rendering, and artifact writing. The implementation stays deterministic and test-first, with new sidecar/output behavior added without redesigning the CLI.

**Tech Stack:** Python 3.13+, `beautifulsoup4`, `pydantic`, `pytest`, `ruff`, `uv`

---

## File Structure

- `src/wiki2md/document.py`: add infobox data models and attach them to `Document`
- `src/wiki2md/models.py`: widen `cleanup_stats` typing so metadata can carry `has_infobox`
- `src/wiki2md/normalize.py`: parse infobox rows, improve image caption/alt fallback, and keep body extraction stable
- `src/wiki2md/assets.py`: select the infobox image from `document.infobox` instead of relying on a body block
- `src/wiki2md/render_markdown.py`: render the infobox image below the title and a `## Profile` section before the lead summary
- `src/wiki2md/writer.py`: write `infobox.json` alongside the existing artifact set

- `tests/fixtures/html/person_fragment.html`: enrich the English fixture with real infobox fields
- `tests/fixtures/html/person_fragment_zh.html`: add a Chinese-compatible infobox fragment

In `src/wiki2md/normalize.py`, add link-aware infobox extraction:

```python
def _extract_infobox_links(node: Tag, article: FetchedArticle) -> list[InfoboxLink]:
    links: list[InfoboxLink] = []
    for anchor in node.find_all("a", href=True):
        text = _clean_text(anchor)
        href = anchor.get("href", "")
        if not text or not href or href.startswith("#"):
            continue
        links.append(InfoboxLink(text=text, href=_normalize_href(article, href)))
    return links


def _extract_infobox(article: FetchedArticle, infobox: Tag, title: str) -> InfoboxData:
    image_block = _extract_image_block(infobox, role="infobox")
    image = _build_infobox_image(image_block, title)
    fields: list[InfoboxField] = []
    for row in infobox.select("tr"):
        label_node = row.find("th")
        value_node = row.find("td")
        if label_node is None or value_node is None:
            continue
        label = _clean_text(label_node)
        text = _clean_prose_text(value_node)
        if not label or not text:
            continue
        fields.append(
            InfoboxField(
                label=label,
                text=text,
                links=_extract_infobox_links(value_node, article),
            )
        )
    return InfoboxData(title=title, image=image, fields=fields)
```

```python
def _build_infobox_image(block: ImageBlock | None, article_title: str) -> InfoboxImage | None:
    if block is None:

    caption = block.caption or block.alt or article_title
    alt = block.alt or caption or article_title
    return InfoboxImage(title=block.title, path=None, alt=alt, caption=caption)
```

```python
def _render_profile(document: Document) -> list[str]:
    if document.infobox is None or not document.infobox.fields:
        return []

    lines = ["## Profile", ""]
    for field in document.infobox.fields:
        lines.append(f"- {field.label}: {field.text}")
    lines.append("")
    return lines


def render_markdown(document: Document, metadata: ArticleMetadata, asset_map: dict[str, str]) -> str:
    lines = [_render_frontmatter(metadata), "", f"# {document.title}", ""]

    if document.infobox and document.infobox.image:
        relative_path = asset_map.get(document.infobox.image.title)
        if relative_path:
            lines.append(f"![{document.infobox.image.alt}](./{relative_path})")
            if document.infobox.image.caption:
                lines.append(f"*{document.infobox.image.caption}*")
            lines.append("")

    lines.extend(_render_profile(document))

    for paragraph in document.summary:
        lines.append(paragraph)
        lines.append("")
    ...
```
